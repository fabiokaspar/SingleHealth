Nome do sistema

 - SingleHealth

Descrição breve

O SingleHealth deve unificar o sistema de saúde brasileiro (hospitais particulares e públicos, clínicas médicas, PS, UBS, AMA...) com intuito de armazenar dados de consultas/exames médicos e laboratoriais dos pacientes. O sistema oferecerá suporte a várias bandeiras de hospitais, as quais possuem várias unidades geograficamente espalhadas pelo território nacional. O sistema indicará aos pacientes as especializações médicas em cada uma das unidades, além das eventuais informações de interesse público. 




Além disso, o sistema pode eventualmente armazenar dados sobre:

- as unidades cadastradas (localização, especializações médicas, tipos de exame oferecidos, qualidade médico-hospitalar, valor, horarios de atendimento, ...)
- pacientes das unidades de hospitais cadastrados (resultados e histórico de exames, consultas marcadas, medicação usada,... )
- profissionais envolvidos nas consultas, como enfermeiros, médicos, secretárias, ...
- estoque de produtos de enfermaria, de limpeza, de higiene, ...
