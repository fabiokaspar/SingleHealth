Nome do sistema

 - SingleHealth

Descrição breve

O sistema SingleHealth deve unificar unidades informatizadas de hospitais particulares e 
públicos no país com intuito de servir como um sistema de recomendação para pacientes que precisam de um determinado serviço médico de maneira rápida, 
assim como cadastrar seus dados hospitalares.
Também deve facilitar a organização estrutural das unidades, dos médicos, enfermeiros e funcionários em geral.

O sistema deve armazenar dados sobre:

- as unidades cadastradas (localização, especializações médicas, tipos de exame oferecidos, qualidade médico-hospitalar, valor, horarios de atendimento, ...)
- pacientes das unidades de hospitais cadastrados (resultados e histórico de exames, consultas marcadas, medicação usada,... )
- profissionais envolvidos nas consultas, como enfermeiros, médicos, secretárias, ...
- estoque de produtos de enfermaria, de limpeza, de higiene, ...
